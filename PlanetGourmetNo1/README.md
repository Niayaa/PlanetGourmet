#### 项目介绍

<p style="padding:10px;"  width="90%">
  什么是美食星球！
  目前暂定为"饮食江山-饮食星球"，由于想名字是一件非常复杂的事情，项目暂定为饮食江山，老土就老土吧！
  以后再说以后的！

  饮食江山是一个以中华美食文化，发展，环境为基点的一个美食类项目，涵盖了中国5000年的美食文化，以此为项目核心，不断发掘历史，宣传中华美食为己任的学习计划。

  

</p>
